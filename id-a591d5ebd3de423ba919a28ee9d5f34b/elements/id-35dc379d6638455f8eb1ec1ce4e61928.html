CU07 - identifier le meuble ()

Identifier un produit ou un composant
Fiche Suiveuse

Permet de retrouver les informations de référence du meuble ainsi que les données relatives à la commande de vente. Une vignette représente dans certains cas le meuble contenu dans le carton. Compte-tenu du fait que la fiche suiveuse suit le produit jusque chez le client, il faudrait que les différents libellés et descriptions soient traduits dans la langue du client. Cela signifie que les fonctionnalités d'identification du produit ou de guidance destinées à l'interne (opérateur de production) doivent être assurées par un autre dispositif (écran). La fiche suiveuse n'est plus "auto porteuse" de la fonctionnalité dans le cas d'une traduction en langue étrangère.
